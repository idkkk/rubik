# Rubik
Rubik is a rapid develop framework.

## Features
1. Modular
2. Easymaintenance
3. Flexibility
4. Friendly

### Frontend
1. react, falcor, webpack
2. redux
3. jQuery, jQuery UI
4. bootstrap

### Backend
1. Spring Boot
2. Spring Security / Shiro
3. Spring JPA / Jdbc
4. Spring Cache(Redis, Guava)
5. JSON / Protobuf
6. Activiti

### Business
1. Authentication, Authorization
2. Implement RBAC base Party Pattern
3. Implement workflow base Activiti

### Functions
1. Database Connection Pool
2. Global Exception Handler
3. Logger
4. Transaction Manager
5. Change Theme
6. Rich Client / Cross Browser
